# 🌦 Weather Telegram Bot (GitHub Actions)

Простой Python-скрипт для автоматической отправки текущей погоды в Telegram-канал. Работает на базе **Open-Meteo API** и запускается бесплатно через **GitHub Actions**.

## 🚀 Функции
* **Данные:** Температура, влажность, скорость и направление ветра.
* **Астрономия:** Время восхода и заката.
* **Автоматизация:** Запуск по расписанию (cron) или при пуше в репозиторий.
* **Уведомления:** Красивая HTML-разметка и "тихий" режим (без звука).

## 🛠 Настройка (GitHub Secrets)
Чтобы скрипт заработал в GitHub Actions, добавь следующие секреты в `Settings -> Secrets and variables -> Actions`:

| Секрет | Описание |
| :--- | :--- |
| `LATITUDE` | Широта (например, `36.58`) |
| `LONGITUDE` | Долгота (например, `15.80`) |
| `TELEGRAM_TOKEN` | Токен вашего бота от @BotFather |
| `TELEGRAM_CHAT_ID` | ID канала или чата (например, `-100...`) |

## 📦 Стек
`Python 3.10+`, `requests`, `openmeteo-requests`, `pandas`.

---

### Как запустить локально
1. Создай файл `.env` с теми же переменными, что указаны выше.
2. Установи зависимости: `pip install -r requirements.txt`.
3. Запусти: `python app.py`.
